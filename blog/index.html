November 2020 - Off-Road Fabrication Website With AWS!

CEFAB - Off-Road Fabrication Website, Powered by AWS!

Over the last month I have spent a ton of time on a new web development project, the CEFAB Off-Road website. This is a personal project that I have created to showcase some of my ongoing off-road related fabrication projects! Check out the WJ Cherokee section, it's the first of many project pages to come!
I also spent some time learning the basics of Amazon Web Services with the help of the extensive documentation available. The end result is a static website hosted through AWS Amplify, with SSL, that is connected to my GitHub repository, and updates automatically every time I push a new update to the repo. In short, I have found that AWS provides a more efficient way to host a website, with extensive and accurate documentation to help you get it hosted, and is a much more affordable alternative over traditional hosting services!

Moving Forward

Over the next month I intend to spend more time learning the AWS platform, as I have only scratched the surface with this project over the last month. I plan to learn more about APIs and will use them to incorporate some new and more interactive features into all of the websites I have created!

September 2020 - Blackjack is Live!

BlackJack - My first tutorial free project is live!
I have been working through a few of the many available online resources to learn some of the basics of programming, most notably freeCodeCamp, codewars, and codecademy pro. These three tools have proven essential as I began my coding journey.
I am proud to announce that my very first project I have ever created using HTML, CSS, and Vanilla JS is live. It is a basic blackjack game, and every part of this game was created without the help of any online tutorials. I have always believed the best way to learn something is to just start doing it, and programming is proving to be no exception!
The game is pretty basic for now, but it is completely functional and it really helped me understand some of the practial uses of JavaScript classes, function parameters, if / else statements, and for loops. It also gave me a greater understanding of the DOM and how to manipulate objects within it, and I am quickly realizing how powerful of a tool that can be.

Moving Forward

My primary focus at this time - learning the basics of JavaScript in a web development environment. Moving forward, I will continue to develop simple applications as I learn more about basic JavaScript, and once I have a greater understanding of Vanilla JS, I will move to frameworks - which I am definitely excited to learn how to use!
